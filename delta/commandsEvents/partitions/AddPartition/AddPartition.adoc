[[cmd-AddPartition]]
===== Add partition
Add <<AddPartition.newPartition>> node as <<new-nodes, new>> partition.

The client implicitly subscribes to any changes to <<PartitionAdded.newPartition>> or any of its descendants.

.Technical name
`AddPartition`

[horizontal]
.Parameters
[[AddPartition.newPartition, `newPartition`]]newPartition:: <<singleChunkType>>
Subtree to form a new partition.
The _anchor node_ MUST NOT contain a _parent_ (as this is a partition, thus cannot have a parent).
All nodes in that subtree MUST be <<new-nodes, new nodes>>.
+
NOTE: This parameter differs from the <<PartitionAdded.newPartition, corresponding event parameter>>:
The command's parameter contains a whole subtree, whereas the event's parameter may contain only the top of the subtree.{fn-org425}
