[[evnt-AnnotationAdded]]
===== Annotation added
<<new-nodes, New node>> <<AnnotationAdded.newAnnotation>> has been added to <<AnnotationAdded.parent>>'s annotations at <<AnnotationAdded.index>>.

All other annotations inside <<AnnotationAdded.parent>>'s annotations with index >= <<AnnotationAdded.index>> have been moved to next higher index.

.Technical name
`AnnotationAdded`

[horizontal]
.Parameters
[[AnnotationAdded.parent, `parent`]]parent:: <<targetNodeType>>

[[AnnotationAdded.newAnnotation, `newAnnotation`]]newAnnotation:: <<singleChunkType>>
Subtree that formed the new annotation.
The _anchor node_ MUST contain the <<AnnotationAdded.parent>> as _parent_.{fn-org357}
All nodes in that subtree MUST be <<new-nodes>>.{fn-org326}
