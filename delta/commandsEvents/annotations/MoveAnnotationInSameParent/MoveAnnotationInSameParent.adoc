[[cmd-MoveAnnotationInSameParent]]
===== Move annotation in same parent
Move <<existing-nodes, existing node>> <<MoveAnnotationInSameParent.movedAnnotation>>
that’s initially at index <<MoveAnnotationInSameParent.oldIndex>>
in <<MoveAnnotationInSameParent.parent>>'s annotations
by <<MoveAnnotationInSameParent.indexOffset>> positions
to index (<<MoveAnnotationInSameParent.oldIndex>> + <<MoveAnnotationInSameParent.indexOffset>>).

This conforms to <<justMovingANode, the action of moving a node in one specific list>> — see that section for more information, diagrams, and examples.
