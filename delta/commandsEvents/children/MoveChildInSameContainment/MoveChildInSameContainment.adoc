[[cmd-MoveChildInSameContainment]]
===== Move child in same containment
Move <<existing-nodes, existing node>> <<MoveChildInSameContainment.movedChild>>
that’s initially at index <<MoveChildInSameContainment.oldIndex>>
within the <<MoveChildInSameContainment.containment>> inside <<MoveChildInSameContainment.parent>>
by <<MoveChildInSameContainment.indexOffset>> indices
to index (<<MoveChildInSameContainment.oldIndex>> + <<MoveChildInSameContainment.indexOffset>>).

This conforms to <<justMovingANode, the action of moving a node in one specific list>> — see that section for more information, diagrams, and examples.
