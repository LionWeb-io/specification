[[evnt-ChildMovedInSameContainment]]
===== Child moved in same containment
<<existing-nodes, Existing node>> <<ChildMovedInSameContainment.movedChild>>
that was initially at index <<ChildMovedInSameContainment.oldIndex>>
within the <<ChildMovedInSameContainment.containment>> inside <<ChildMovedInSameContainment.parent>>
is moved by <<ChildMovedInSameContainment.indexOffset>> indices
to index (<<ChildMovedInSameContainment.oldIndex>> + <<ChildMovedInSameContainment.indexOffset>>).

This conforms to <<justMovingANode, the action of moving a node in a single list>> — see that section for more information, diagrams, and examples.
