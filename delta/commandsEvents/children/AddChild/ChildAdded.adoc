[[evnt-ChildAdded]]
===== Child added
<<new-nodes, New node>> <<ChildAdded.newChild>> has been added to <<ChildAdded.parent>>'s  <<ChildAdded.containment>> at <<ChildAdded.index>>.

All other children inside <<ChildAdded.parent>>'s <<ChildAdded.containment>> with index >= <<ChildAdded.index>> have been moved to next higher index.
