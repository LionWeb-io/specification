[[evnt-ChildAdded]]
===== Child added
<<new-nodes, New node>> <<ChildAdded.newChild>> has been added to <<ChildAdded.parent>>'s  <<ChildAdded.containment>> at <<ChildAdded.index>>.

All other children inside <<ChildAdded.parent>>'s <<ChildAdded.containment>> with index >= <<ChildAdded.index>> have been moved to next higher index.

.Technical name
`ChildAdded`

[horizontal]
.Parameters
[[ChildAdded.parent, `parent`]]parent:: <<targetNodeType>>

[[ChildAdded.newChild, `newChild`]]newChild:: <<singleChunkType>>
Subtree that formed the new child.
The _anchor node_ MUST contain the <<ChildAdded.parent>> as _parent_.{fn-org357}
All nodes in that subtree MUST be <<new-nodes>>.{fn-org326}
