[[cmd-ChangeReferenceTarget]]
===== Change reference target
Change <<ChangeReferenceTarget.oldTarget>> of existing entry inside <<ChangeReferenceTarget.parent>>'s <<ChangeReferenceTarget.reference>> at <<ChangeReferenceTarget.index>> to <<ChangeReferenceTarget.newTarget>>.

****
Current ResolveInfo MAY be `null` as we can only change target if both current Target and <<ChangeReferenceTarget.newTarget>> is non-`null`.
****
