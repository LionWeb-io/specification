[[cmd-ChangeReferenceResolveInfo]]
===== Change reference ResolveInfo
Change <<ChangeReferenceResolveInfo.oldResolveInfo>> of existing entry inside <<ChangeReferenceResolveInfo.parent>>'s <<ChangeReferenceResolveInfo.reference>> at <<ChangeReferenceResolveInfo.index>> to <<ChangeReferenceResolveInfo.newResolveInfo>>.

****
Current target MAY be `null` as we can only change ResolveInfo if both current ResolveInfo and <<ChangeReferenceResolveInfo.newResolveInfo>> is non-`null`.
****
