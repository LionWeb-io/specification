Add partition

Add newPartition node as new partition.

Before After

nothing
Parameters
newPartition

[chunkType]{fn-org327} root node to form a new partition. The root node MUST NOT contain a parent (as this is a partition, thus cannot have a parent). All nodes in that subtree MUST be new, i.e. their id MUST NOT exist in the repository. Nodes in that subtree MAY have references to already existing nodes, and already existing nodes MAY have references to nodes in that subtree.{fn-org327}

Composite

Groups several commands into a logical group.{fn-org281} The parts are ordered.

Command groups don’t guarantee anything; the repository MAY take the group as a hint to resolve conflicts in a different manner.

Composite commands have a command id on their own.{fn-org353} We can nest composite commands arbitrarily.{fn-org360}
