[[qry-SubscribeToChangingPartitions]]
===== Subscribe to partition changes
This client wants to receive <<evnt-PartitionAdded>> events on newly created (if <<SubscribeToChangingPartitions.creation>> is true)
or <<evnt-PartitionDeleted>> events on deleted (if <<SubscribeToChangingPartitions.deletion>> is true) partitions.
The client automatically subscribes to newly created partitions (if <<SubscribeToChangingPartitions.creation>> is true), and receives the full contents of partitions.

See also <<qry-InformAboutChangingPartitions>>.

.Technical name
`SubscribeToChangingPartitions`

[horizontal]
.Request parameters
[[SubscribeToChangingPartitions.creation, `creation`]]creation:: <<booleanType>>
Whether this client wants to receive events on newly created partitions (`true`), or not (`false`).

[[SubscribeToChangingPartitions.deletion, `deletion`]]deletion:: <<booleanType>>
Whether this client wants to receive events on deleted partitions (`true`), or not (`false`).
