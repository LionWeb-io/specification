[[qry-ListPartitions]]
===== List partitions
Lists all non-language partitions accessible in the repository.{fn-org361}

Same functionality as <<{bulk}.adoc#listPartitions, bulk API listPartitions command>>.

See also <<qry-ListAndSubscribePartitions>>.
