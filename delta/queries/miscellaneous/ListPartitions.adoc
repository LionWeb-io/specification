[[qry-ListPartitions]]
===== List partitions
Lists all non-language partitions accessible in the repository.{fn-org361}

Same functionality as <<{bulk}.adoc#listPartitions, bulk API listPartitions command>>.

.Technical name
`ListPartitions`

[horizontal]
.Request parameters
[[ListPartitions.requestQueryId, `queryId`]]queryId:: <<queryIdType>>
Id of this query.

[[ListPartitions.requestAdditionalInfos, `additionalInfos`]]additionalInfos:: <<additionalInfosType>>

[horizontal]
.Response parameters
[[ListPartitions.partitions, `partitions`]]partitions:: <<chunkType>>
All accessible <<{m3}.adoc#partition, Partitions>> in the Repository.
The partitions are sent as complete nodes.
Does NOT include any children or annotations of the root partition nodes.
Does NOT include <<{m3}.adoc#Language, Languages>> or partition children/annotations.
